D₀ 차이 (Poisson 기준)
—
⚠ n이 작을수록 B-E D₀ 값이 Poisson D₀보다 크게 높아집니다.
이는 B-E 모델이 결함 뭉침을 보정하기 위해 더 높은 결함밀도를 할당하기 때문입니다.
Poisson Model
Y = exp(−D₀ · A)
→ D₀ = −ln(Y) / A
⇄
Bose-Einstein Model
Y = 1 / (1 + D₀·A)n
→ D₀ = (Y−1/n − 1) / A
◆ n-factor 란?
n = 1
결함이 완전히 뭉친 상태 (극단적 클러스터링)
B-E 단순형: Y = 1/(1+D₀A)
1 < n < ∞
현실적인 부분 클러스터링 상태
제조사별, 공정별로 피팅
n → ∞
결함이 완전 랜덤 분포 → Poisson과 동일
lim B-E = Poisson (수학적 증명)
◆ n 과 레이어(Layer) 수의 관계
Layer 수 = n 의 물리적 해석
Bose-Einstein 모델에서 n의 가장 직관적인 물리적 의미는 웨이퍼 제조에 관여하는 포토리소그래피 마스크(레이어) 수입니다.

아이디어는 간단합니다: 각 레이어마다 독립적으로 결함이 Poisson 분포를 따른다면, n개 레이어를 모두 통과해야 하는 칩의 총 Yield는
Y = Y_per_layer^n = (e^(−D₀_layer·A))^n 이 됩니다. 이를 B-E 형태로 쓰면 n이 레이어 수와 대응됩니다.

⚠ 주의사항
실제 fab에서는 n을 레이어 수와 정확히 같게 쓰지 않습니다.
n은 실측 Yield 데이터를 곡선 피팅(curve fitting)해서 결정하는 경험적 파라미터입니다.
레이어 수는 출발점 추정값으로 활용하고, 실제 값은 fab마다 다릅니다.

◆ 변환 수식 완전 정리
Poisson → B-E (n 포함)
1. Y 계산: Y = exp(−D₀_P · A)
2. B-E 역산: D₀_BE = n · (Y−1/n − 1) / A
= n · (exp(D₀_P · A / n) − 1) / A  로도 표현 가능
B-E → Poisson (n 포함)
1. Y 계산: Y = (1 + D₀_BE · A / n)−n
2. Poisson 역산: D₀_P = −ln(Y) / A
= ln(1 + D₀_BE · A / n) · n / A
n → ∞ 극한 증명
lim(n→∞) (1 + D₀·A/n)−n = e−D₀·A
n이 무한히 커지면 B-E가 Poisson과 수렴 → n이 클수록 두 D₀ 수치 차이 줄어듦
실무 요점: D₀를 공유할 때 반드시 모델명 + n값을 함께 명시해야 합니다.
