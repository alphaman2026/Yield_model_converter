변환 수식 정리
공통 기반  동일 Y, 동일 A에서 D₀만 역산 → 완전 가역 변환
Poisson → B-E   D₀BE = (eD₀P·A − 1) / A   ← Y = e−D₀A 를 1/(1+D₀A)에 대입
B-E → Poisson   D₀P = ln(1 + D₀BE·A) / A   ← Y = 1/(1+D₀A) 를 e−D₀A에 대입
⚠ 두 D₀는 같은 Y를 예측하지만 수치는 다름: D₀BE > D₀P (항상 성립)
